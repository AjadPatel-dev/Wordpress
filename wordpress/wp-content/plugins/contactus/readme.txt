== Installation ==

Adding Contactus to your Wordpress site is really easy, follow these steps :

* Install and activate the Contactus Plugin through the 'Plugins' menu in WordPress.
* After installation, click the Contactus section in the left toolbar.
* Customize your <a href="https://contactus.nikba.com/?utm_campaign=wordpress_plugin&utm_medium=widget&utm_source=wordpress" target="_blank">Contactus</a> and copy the code.
* Go back to Wordpress console, paste the code and press Save changes.
* Done!
